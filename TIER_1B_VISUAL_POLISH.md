# Tier 1B — Visual Polish Pass

This patch builds on the research clarity patch and the Duelyst integration branch.

## Goals

- Make imported animated creature GIFs feel intentionally integrated.
- Improve readability and presentation of creature cards.
- Fix resource icon scaling.
- Reduce visual confusion from small imported icon artifacts.
- Keep the project lightweight and GitHub Pages friendly.

## Changes

- Enlarged animated creature display area.
- Scaled creature GIFs up so they are the focal point of each card.
- Strengthened selected-card visual feedback.
- Improved resource bar icon sizing and alignment.
- Added subtle Genesis-style panel lighting.
- Improved egg preview display.
- Added clearer visual treatment for trait/mutation icons.
- Added a startup log note when the Tier 1B polish is active.

## Notes

This is still a visual polish pass, not a full UI rewrite. The next deeper pass would replace the CSS panels/buttons with fully sliced UI assets and a dedicated facility-console layout.
