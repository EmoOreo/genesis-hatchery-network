# Tier 1 Asset Pass

This build integrates a small, curated set of uploaded Tier 1 assets into the Hatchery Network prototype.

## Included

### Kenney UI Pack Space Expansion
Used for UI/progress-bar-ready assets. Font files were intentionally excluded.

### Kenney Particle Pack
Used for hatch, mutation, anomaly, research, and transfer visual effects.

### Glitch Client PNG Icons
Used only for small resource/interface icons:
- DNA crystal
- Food/action icon
- Research data bolt
- Archive check
- Upgrade badge
- Slot frame icons

## What Changed

- Added asset folders under `assets/`.
- Added resource icons to DNA/Food/Data counters.
- Added particle burst layer for hatching, breeding, mutation, transfer, and research events.
- Added incubation progress bars.
- Added visual selected-card emphasis.
- Added a lightweight asset attribution/license folder.

## Not Included

- No font files.
- No large Glitch FLA/SWF source files.
- No full Glitch client/server source code.
- No giant sprite dump import.

## Next Tier 1 Improvement

The next upgrade would be deeper UI reskinning: replacing the current CSS buttons/panels with sliced Kenney UI panels and building a dedicated facility console layout.
